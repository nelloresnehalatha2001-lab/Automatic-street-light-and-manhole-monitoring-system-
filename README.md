# Automatic Street Light and Manhole Monitoring System

### 📝 Description
This project aims to automate street lighting based on surrounding light levels and monitor manhole status to prevent accidents.

### 🚀 Features
*   **Smart Lighting:** Automatically turns ON at night and OFF during the day using LDR sensors.
*   **Manhole Monitoring:** Detects if a manhole cover is open or if water levels are rising.
*   **Real-time Alerts:** Sends notifications to authorities.

### 🛠️ Components
*   Arduino Uno / NodeMCU
*   LDR Sensor
*   Ultrasonic Sensor
*   LEDs & Relays
